    French Intent Classification I built a French Intent Classification application using pre-trained HuggingFace Transformers models (FlauBERT, CamemBERT) as word/sentence embeddings, and simple ML algorithms (Logistic Regression, MLP) as classificators. The CLINC150 training and validation sets were sampled to only keep the classes of interest, and translated to french before training the classificators. Multiple methods were implemented and are easy to switch between, thanks to a configuration file. I obtained an average F1-Score of 98% on the test set with the best model (CamemBERT Large + Logistic Regression).
