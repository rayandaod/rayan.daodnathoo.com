My projects
Spoteezer is a web application allowing to convert a Deezer link into a Spotify link and vice-versa. It is now less a hustle (at least for me) to share music from Spotify to family and friends that are on Deezer :) I built it using HTML/CSS for the front-end and a Flask Python server as back-end. The latter communicates with the API of both streaming platforms to process the input Deezer link and provide the best Spotify match according to the information retrieved. Try it now!
